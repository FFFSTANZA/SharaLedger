Banking and Statements

Managing your bank transactions in Versoll Books is designed to be straightforward. Instead of manual entry for every transaction, you can import your bank statements and categorize them quickly.

Bank Statement Import

Versoll Books supports CSV statement imports from almost any bank. The system uses intelligent column detection to map your statement fields to the correct data points.

Step 1

Upload Statement

Navigate to the Banking section and click on 'Import Statement'. Select the CSV file exported from your bank's net banking portal.

Step 2

Column Mapping

The system will attempt to automatically detect columns for Date, Description, Reference, and Amount. If it fails or misidentifies a column, you can manually select the correct headers from the dropdown menus.

Step 3

Review Transactions

Once imported, transactions appear in the Unreconciled list. Each record displays the raw bank narration, date, and amount.

Automatic Transaction Categorization

Versoll Books features intelligent automatic transaction categorization that dramatically reduces manual work. The system uses sophisticated pattern matching with confidence scoring to categorize your bank transactions automatically.

Smart Categorization Features

Pattern-Based Recognition

The system recognizes 20+ transaction patterns automatically:

Bank Transfers

NEFT, IMPS, RTGS, UPI, online transfers

Confidence: 90%+

Utilities

Electricity, water, gas, internet, phone bills

Confidence: 80%+

Salary & Payroll

Salary payments, wages, employee payments

Confidence: 90%+

Tax Payments

GST, TDS, income tax payments

Confidence: 90%+

Rent & Property

Rent payments, lease payments

Confidence: 80%+

Insurance

Insurance premiums, life insurance

Confidence: 80%+

Vendor Payments

Supplier payments, purchase expenses

Confidence: 70%+

Customer Receipts

Customer payments, sales receipts

Confidence: 80%+

Interest Income

Bank interest, dividends, ROI

Confidence: 90%+

Loan EMI

Home loan, personal loan EMIs

Confidence: 80%+

Confidence Scoring System

Each categorization includes a confidence score:

  • High Confidence (90%+): Exact pattern match - very likely correct
  • Medium Confidence (70-89%): Keyword match - likely correct but verify
  • Low Confidence (<70%): General categorization - review needed

Automatic Document Type Detection

Based on transaction direction and patterns, the system automatically suggests:

  • Payment Entry - For money going out (vendor payments, expenses)
  • Receipt Entry - For money coming in (customer payments, income)
  • Journal Entry - For adjustments (bank charges, transfers)

Duplicate Prevention

Versoll Books automatically prevents duplicate imports using a sophisticated hashing algorithm:

  • Generates unique hash based on: date + amount + description + bank account + reference
  • Detects duplicates within the same CSV file
  • Prevents re-importing previously imported transactions
  • Shows duplicate indicators in import preview

Manual Categorization (When Needed)

For transactions that need human review, you can manually adjust:

  • Party - Select the customer or supplier
  • Account - Choose the specific ledger account
  • Voucher Type - Override automatic suggestion

Key Benefit: Time Savings

With automatic categorization, most transactions are pre-filled with accurate suggestions. For typical businesses, this reduces reconciliation time by 60-80% compared to manual entry.

Bank Accounts Management

You can manage multiple bank accounts and credit cards within Versoll Books. Each account is linked to a corresponding ledger in your Chart of Accounts, allowing for accurate financial reporting.
